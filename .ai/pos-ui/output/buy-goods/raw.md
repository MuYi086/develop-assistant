# 购买商品 - 结构化需求

## 功能概述
实现门店商品选购功能，包括商品列表展示、规格选择、购物车管理、订单提交等核心流程。

## 用户故事

### US-001: 浏览在售商品
作为收银员，我希望看到当前门店所有在售商品，以便向顾客推荐。

**验收标准**：
- [ ] 展示"上架中"状态的商品列表
- [ ] 列表字段：商品图片、商品名称、原价、在售数量、商品类型（单品/套餐）、已加购数量、商品状态（在售/已售罄）
- [ ] 左侧 Tab 切换商品分类
- [ ] 支持商品名称模糊搜索
- [ ] 可隐藏/显示售罄商品
- [ ] 以后端接口返回顺序为准，前端直接遍历显示

### US-002: 商品分类筛选
作为收银员，我希望按分类筛选商品，以便快速找到特定类型商品。

**验收标准**：
- [ ] 左侧 Tab 栏展示商品分类
- [ ] 点击 Tab 切换当前分类
- [ ] 筛选后列表即时更新

### US-003: 搜索商品
作为收银员，我希望通过名称搜索商品，以便快速定位。

**验收标准**：
- [ ] 点击"搜索"按钮弹出搜索框
- [ ] 支持商品名称模糊搜索
- [ ] 搜索结果实时展示

### US-004: 加购单品或套餐
作为收银员，我希望将商品加入购物车，以便后续结算。

**验收标准**：
- [ ] 点击【单品】或【无可选规格的套餐】直接加购
- [ ] 点击【有可选规格的套餐】弹出规格选择弹窗
- [ ] 在规格弹窗中选择规格后点击"添加到购物车"完成加购
- [ ] 加购成功后关闭弹窗，购物车新增记录

### US-005: 管理购物车
作为收银员，我希望查看和管理已加购商品，以便确认订单内容。

**验收标准**：
- [ ] 展示待结算商品清单
- [ ] 列表字段：商品/SKU名称、子商品明细（套餐）、售价、加购数量
- [ ] 以后端接口返回的购物车列表为准，前端直接遍历显示
- [ ] 支持增减商品数量
- [ ] 支持直接输入数量
- [ ] 支持清空购物车（需弹窗确认）

### US-006: 提交购物车
作为收银员，我希望提交购物车进入订单确认，以便完成收款。

**验收标准**：
- [ ] 购物车为空时"去收款"按钮置灰不可点击
- [ ] 点击"去收款"提交购物车
- [ ] 提交成功后进入"确认订单页"

### US-007: 切换用餐方式
作为收银员，我希望切换就餐/外带模式，以便适应不同场景。

**验收标准**：
- [ ] 支持切换用餐方式
- [ ] 切换时检查购物车商品可用性
- [ ] 若有不兼容商品，弹窗二次确认："是否确认切换用餐方式？切换后，已加购的xx将被移除"
- [ ] 确认后才执行切换并移除不兼容商品

### US-008: 重置商品数据
作为收银员，我希望重置本机商品数据，以便同步最新云端数据。

**验收标准**：
- [ ] 点击"重置本机商品"弹出二次确认
- [ ] 确认后向云端请求最新商品数据
- [ ] 重置本地商品档案

## 异常处理

| 场景 | 触发条件 | 处理方式 |
|------|----------|----------|
| 离店云端模式 | 网络模式为"离店云端模式" | 无法使用本功能 |
| 查询商品列表异常 | 无内容 | 界面显示"请更换筛选条件" |
| 查询商品超时 | 请求超时 | 界面显示"获取数据超时" |
| 查询商品失败 | 后端返回错误 | 界面显示"查询商品失败" |
| 提交购物车超时 | 请求超时 | Toast提示"提交购物车超时" |
| 提交购物车失败 | 后端返回错误 | 弹窗提示"后端返回的错误原因" |

## 数据类型定义

```typescript
// 商品
interface Product {
  id: string;
  name: string;
  originalPrice: number;
  salePrice?: number;
  image: string;
  stockQuantity: number;
  categoryId: string;
  type: 'single' | 'set'; // 单品或套餐
  hasSpecs: boolean; // 是否有可选规格
  status: 'on_sale' | 'sold_out';
  cartQuantity?: number; // 已添加到购物车的数量
}

// 套餐规格
interface SetSpec {
  id: string;
  name: string;
  subItems: SubItem[];
}

// 子商品
interface SubItem {
  productId: string;
  productName: string;
  quantity: number;
}

// 购物车项
interface CartItem {
  id: string;
  productId: string;
  productName: string;
  specId?: string;
  specName?: string;
  subItems?: SubItem[];
  price: number;
  quantity: number;
}

// 购物车
interface ShoppingCart {
  items: CartItem[];
  totalAmount: number;
  totalCount: number;
  diningMode: 'dine_in' | 'takeout';
}
```
